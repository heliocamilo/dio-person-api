# dio-person-api
<h2>API de Gerenciamento de Pessoas - Bootcamp Digital Innovation One</h2>

Este código é uma extensão da Live Coding de desenvolvimento de um pequeno sistema para o gerenciamento de pessoas, com objetivo de introduzir as práticas de desenvolvimento de API REST utilizando Spring Boot.
Referência: https://github.com/rpeleias/personapi_digital_innovation_one

Endereço de execução da API na nuvem:

https://hcs-dio-person-api.herokuapp.com/

Foram utilizados:

* Setup inicial de projeto com o Spring Boot Initialzr 
* Criação de modelo de dados para o mapeamento de entidades em bancos de dados
* Desenvolvimento de operações de gerenciamento de pessoas (Cadastro, leitura, atualização e remoção de pessoas de um sistema).
* Relação de cada uma das operações acima com o padrão arquitetural REST, e a explicação de cada um dos conceitos REST envolvidos durante o desenvolvimento do projeto.
* Desenvolvimento de testes unitários para validação das funcionalidades
* Implantação do sistema na nuvem através do Heroku

Para executar o projeto no terminal, digite o seguinte comando:

```shell script
mvn spring-boot:run 
```

Após executar o comando acima, basta apenas abrir o seguinte endereço e visualizar a execução do projeto:

```
http://localhost:8080/api/v1/people
```


Tecnologias utilizadas:

* Java 8 ou versões superiores (a versão original foi feito em Java 11).
* Map Struct 1.4.1
* Lombok 1.18.20
* Spring Fox 2.9.2 - Swagger
* Maven 3.6.3 ou versões superiores.
* Intellj IDEA Community Edition ou sua IDE favorita.
* Controle de versão GIT instalado na sua máquina.

* Conta no Heroku para o deploy do projeto na nuvem
